# Source hierarchy

Use this evidence ladder when expanding the corpus.

## Tier 1 — Primary
- Hyperliquid documentation
- Hyperliquid public statements and announcements
- protocol or team postmortems
- onchain / market data
- bug bounty and technical documentation

## Tier 2 — High-quality technical secondary
- security research firms
- technical market-structure analysis
- rigorous exploit writeups

## Tier 3 — News / commentary
- useful for chronology and public reaction
- do not rely on these alone for strong rule design

## Tier 4 — Social posts
- useful for lead generation only
- do not treat these as the final truth
