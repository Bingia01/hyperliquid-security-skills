# Repo refactor notes

This version hardens the repository around evidence hygiene and maintainability.

Key changes:
- moved case studies to a top-level `case_studies/` directory,
- split incident data into `confirmed`, `credible_reports`, and `research_leads`,
- promoted the visual attack taxonomy to `docs/architecture/`,
- added a rendered PNG attack surface map,
- updated the schema and consistency matrix to reflect confidence tiers.
