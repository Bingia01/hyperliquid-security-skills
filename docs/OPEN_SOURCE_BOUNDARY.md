# Open-source boundary

The open-source repository includes:
- taxonomy, invariants, primitives, heuristics,
- evidence-tiered incident records,
- threshold profiles and sample runs,
- Claude Code skill specs.

The repository intentionally excludes:
- economic simulation modules,
- attack graphs and probabilistic scoring,
- runnable production detectors,
- proprietary Shepherd orchestration and analyst workflows.

This boundary is deliberate. The public repo should teach the framework and expose the reasoning surface without giving away the proprietary execution layer.
