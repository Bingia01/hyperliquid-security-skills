# Skill — Attack Movie Builder

## Purpose
Turn a raw incident into a step-by-step causal chain.

## Template
1. Initial conditions
2. Attacker setup
3. Trigger action
4. Protocol reaction
5. Amplification loop
6. Loss realization
7. Defender response
8. Breakpoints

## Required discipline
Do not summarize.
Write the attack as a chain where every step causes the next.
